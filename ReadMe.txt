To run this program from Terminal:
1) Go to src directory. ( For example, Use command “cd src”);
2) Compile all java files use “javac *.java”.
3) Run this program using “java PageReplaceSimulator”. The program will ask you to choose the algorithm name and input_data_file_path. It will run and output the stats after you input them.  


To run this program from IntelliJ:
1.Copy small_data_set.txt and large_data_set.txt to the folder Jingqiu-LRU_FIFO_NRU.
2.open this project (Jingqiu-LRU_FIFO_NRU) in IntelliJ, and run it.
